# Code of Conduct

We aim to maintain an open, constructive, and harassment-free community. By participating in this project you agree to:

- Be respectful and considerate in communications
- Not engage in harassment, discrimination, or personal attacks
- Be open to feedback and collaborate professionally

If you encounter unacceptable behavior, please open an issue marked **security** or contact the maintainers directly.

Thank you for helping us keep this project welcoming and productive.
