# Contributing

Thank you for your interest in contributing to RAG Chatbot! Small, focused contributions are appreciated and make the project better.

Getting started

- Fork the repository and create a feature branch: `git checkout -b feature/your-change`
- Keep changes small and focused; one logical change per commit
- Write clear commit messages and add tests for code changes where applicable

Submitting changes

1. Push your branch to your fork
2. Open a pull request describing the change and why it helps
3. Be responsive to review feedback and iterate as needed

Code style

- Use sensible, consistent formatting and meaningful names
- Use type hints for public functions where appropriate
